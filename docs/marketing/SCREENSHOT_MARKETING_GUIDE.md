## üé® Design Guidelines

### Branding Consistency
- Always use screenshots with consistent UI theme
- Add subtle branded border when needed (5px, brand color)
- Include logo watermark for high-value screenshots

### Annotations
When adding callouts to screenshots:
- Use brand colors (see BRANDING_GUIDE.md)
- Keep arrows subtle and minimal
- Use sans-serif fonts (Inter, SF Pro)
- Maximum 3 callouts per screenshot

### Image Optimization
```bash
# Optimize for web
pngquant --quality=80-95 screenshot.png

# Convert to WebP for better performance
cwebp -q 85 screenshot.png -o screenshot.webp

# Resize for specific platforms
convert screenshot.png -resize 1200x630 screenshot-twitter.png
```
