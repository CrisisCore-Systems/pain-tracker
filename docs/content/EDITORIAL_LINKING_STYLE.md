# Editorial Linking Style

Use the published canonical title when a link appears in a structured context:

- numbered sequences
- bulleted reading paths
- start-here pages
- related-reading blocks that function like navigation

Use shorter contextual anchors in prose when the sentence reads better and the link is not acting as navigation.

Examples:

- Structured list: use `Protective Computing Canon v1.0 is now DOI-backed`
- Running prose: `quality gates that earn trust` is acceptable if the sentence reads naturally

Rules:

1. Preserve the current published URL even if an older title is still technically reachable.
2. Normalize list labels to the current published title when they point at a specific post.
3. Do not mass-rewrite prose just to force canonical titles into every sentence.
4. When a reading-path hub replaces an older generic hub link, use the reading-path page title in list contexts.

This rule exists to keep navigation blocks trustworthy without making the prose stiff or mechanically repetitive.
