# Degraded-Mode Review Scope

Review name: Narrow degraded-mode accessibility review of PainTracker essential journaling path

## Essential path under review

1. Open app.
2. Unlock or reach journaling surface.
3. Create pain entry.
4. Edit entry.
5. View entry.
6. Export or prepare summary.
7. Recover from one failed action.

## Reviewer questions

1. Can this path be completed with keyboard only?
2. Can it be completed with screen reader support?
3. Does low cognition/high pain state make the flow unsafe or confusing?
4. Where does degraded mode fail?
5. Which failure is most harmful?

## Output path

- `docs/reference-implementation/paintracker/external-reviews/degraded-mode-accessibility-review-0001.md`

## Badge impact rule

- Before completed external review: `Self attested` or `Repo backed` only.
- After completed scoped review: may upgrade to `Externally reviewed` for that exact narrow path only.
